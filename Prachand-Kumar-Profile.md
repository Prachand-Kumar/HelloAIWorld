## Nebula Program Understanding (Updated)

### What Nebula Is

Nebula is a **CSS early in profession workforce program** — not an AI agent. It is a structured program to build a new generation of full-time Support Engineers focused on solving lower complexity technical cases, empowered by AI tools including the SME Agent.

Nebula is Customer Zero for CSS's transformation in hiring, training, and developing support talent. It expands support capacity rather than replacing existing expertise.

### What the SME Agent Is

The **SME Agent (Nebula AI Agent / FQR Program)** is the AI-powered tool that empowers Nebula engineers and the broader CSS support workforce. It provides guided troubleshooting (FQRs — Guided Resolution logic) so engineers — especially early in profession — can handle cases faster and with higher consistency.

**Prachand's PM role spans both:** he owns the SME Agent FQR program that enables the Nebula workforce to operate effectively.

### Nebula Workforce Program — Key Facts

- **Purpose:** Build an early in profession FTE workforce for lower complexity commercial case support
- **Not a replacement** for current TSEs — it expands and complements existing capacity
- **Not AI-only** — AI (SME Agent) is an enabler, not a dependency; program validates new workforce structures independently
- **Phase 1 locations:** Mexico City, Pune, Bucharest
- **Reports through** SBU hierarchy like any CSS delivery team — not a separate division
- **Distinct from:** Aspire (rotational/graduate exploration), CCM (case assignment model), Delivery Partners
- **Case model:** Nebula engineers handle a defined subset of commercial cases using a structured complexity model
- **Benefit to senior TSEs:** Frees them for high complexity work, advanced troubleshooting, and escalations
